In order to target bugs bsc#1115644 and bsc#1052042, we want to : - Set static IP and fqdn for first NIC in the list and check /etc/hosts formatting - Open yast2 lan again and change the fqdn, check if /etc/hosts is changed correctly ( bsc#1052042 ) - Set it to DHCP - Set it again to static with new FQDN and check if /etc/hosts is changed correctly ( bsc#1115644 )
